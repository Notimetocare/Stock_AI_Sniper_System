# 💻 Local Engine (本地端資料處理)

本目錄負責系統的「資料收集與初步過濾」。主要工作是每日抓取股價，並透過程式邏輯找出符合特定 K 線型態的標的。

> **註**：涉及個人判斷參數的程式碼不予公開，此處僅展示資料流的處理架構與運算樣本。

---

## 📊 運算流程樣本展示 (Sample Outputs)

為了具體說明程式是如何處理資料的，我在 `sample_outputs/` 目錄下保留了幾檔股票（如台積電 2330、輝達 NVDA 等）的運算結果作為範例：

* **📂 1_Raw_Data**：展示透過 API 抓取下來的原始 K 線 CSV 檔。
* **📂 AI_Support & AI_Pressure**：展示程式如何計算出支撐與壓力數值，並輸出檢驗用的 `_plot` 圖表。
* **📂 Structure**：展示如何將 K 線轉化為可供策略判斷的座標數據 (`structure_v569.csv`)。
* **📂 Daily_Alert_Buy_CSV**：紀錄單一個股觸發條件的歷史軌跡。
* **📂 Daily_Chart & Weekly_Chart**：程式自動繪製並標示進場與防守價位的參考圖表。
* **📂 Daily_Summary_Reports**：本地端每日運算完畢後，輸出的總結清單（供雲端 Bot 讀取推播使用）。
